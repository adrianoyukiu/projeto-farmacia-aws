# Relatório de Projeto: Arquitetura Farmácia Virtual

## 1. Descrição do Problema
Necessidade de uma plataforma de e-commerce para farmácia que suporte picos de acesso e garanta a segurança dos dados dos usuários.

## 2. Solução Proposta
Uma arquitetura totalmente baseada em serviços gerenciados da AWS, eliminando a necessidade de gerenciar servidores físicos e reduzindo custos operacionais.

## 3. Principais Benefícios
- **Segurança:** Uso de criptografia no S3 e DynamoDB.
- **Custo-Benefício:** Modelo de pagamento por uso (Serverless).
- **Escalabilidade:** O sistema cresce automaticamente conforme o número de pedidos aumenta.
